Projeto desenvolvido com foco na criação de dashboards dinâmicos e estratégicos para análise de dados, utilizando Power BI.

🚀 Funcionalidades
Cartões para resumo de métricas
Indicadores de desempenho (KPIs)
Segmentação de dados (filtros interativos)
Gráficos de barras e colunas para comparação
Tabelas com dados detalhados
Mapas com visualização geográfica
Navegação interativa entre páginas (E-commerce, Vendas e Clientes)
Uso de visuais personalizados

📈 Objetivo
Transformar dados em informações claras e relevantes, apoiando a tomada de decisão por meio de visualizações intuitivas.

🧠 Competências Desenvolvidas
Análise de Dados
Power BI
Business Intelligence
Visualização de Dados
Criação de Dashboards
Indicadores de Desempenho (KPIs)
